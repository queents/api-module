# API Generator

Full API CRUD Generator build on repository pattern for VILT Stack

## Install

```bash
composer require queents/api-module
```
Add Module to `modules_statuses.json` if not exists

```json
{
    "API": true
}
```

```bash
php artisan config:clear
```
